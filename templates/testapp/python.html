Python Exam Section

Exam Overview

The Python exam is designed to test your knowledge of Python programming fundamentals, data structures, object-oriented programming, and problem-solving skills.

Exam Structure

  • Duration: 90 minutes
  • Total Questions: 50
  • Question Types: Multiple Choice, Fill in the Blanks, Code Output Prediction, and Coding Problems
  • Passing Score: 70%

Topics Covered

  1. Python Basics - Variables, Data Types, Operators, Input/Output
  2. Control Structures - If-else statements, Loops (for, while), Break and Continue
  3. Functions - Function definition, Arguments, Return values, Lambda functions
  4. Data Structures - Lists, Tuples, Sets, Dictionaries
  5. String Manipulation - String methods, Formatting, Regular expressions
  6. File Handling - Reading and writing files, File operations
  7. Object-Oriented Programming - Classes, Objects, Inheritance, Polymorphism
  8. Exception Handling - Try-except blocks, Custom exceptions
  9. Modules and Packages - Import statements, Standard library modules
  10. List Comprehensions - List, Dictionary, and Set comprehensions

Question Distribution

  • Multiple Choice Questions: 25 questions (1 mark each)
  • Fill in the Blanks: 10 questions (1 mark each)
  • Code Output Prediction: 10 questions (2 marks each)
  • Coding Problems: 5 questions (3 marks each)

Python Exam Specific Rules

  1. Python version used for evaluation: Python 3.x
  2. For coding questions, write clean and efficient code with proper indentation
  3. You may use built-in Python functions and standard library modules
  4. External libraries like NumPy, Pandas are not allowed unless specified
  5. Code will be evaluated based on correctness, efficiency, and readability
  6. Partial marking available for coding questions showing correct logic
  7. Syntax errors will result in zero marks for that particular question
  8. Comments in code are encouraged but not mandatory

Sample Questions

Multiple Choice:

Q. What is the output of: print(type([]))?

a) <class 'tuple'>
b) <class 'list'>
c) <class 'dict'>
d) <class 'set'>

Code Output Prediction:

Q. What will be the output of the following code?

x = [1, 2, 3]
y = x
y.append(4)
print(x)

Coding Problem:

Q. Write a Python function to check if a given string is a palindrome.

Preparation Tips

  • Practice writing code by hand or in a text editor without IDE assistance
  • Focus on understanding concepts rather than memorizing syntax
  • Review common Python built-in functions and their usage
  • Practice tracing code execution and predicting outputs
  • Work on time management - allocate time wisely for each section
  • Review Python documentation for standard library modules
  • Practice coding problems on platforms like LeetCode or HackerRank

Resources for Preparation

  • Official Python Documentation (python.org)
  • Python practice problems and tutorials
  • Previous exam papers and sample questions
  • Online coding platforms for hands-on practice
